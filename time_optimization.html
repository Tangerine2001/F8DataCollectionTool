time_optimization module

time_optimization.close_enough(out1: bytes, out2: bytes) → bool

Checks if two outputs are “close enough” (99% of the characters are the same). This is to account for the slight difference when sending the same command but has milliseconds worth of difference in the output.

Parameters

  • out1 (bytes, Required) – First output. Order does not matter.

  • out2 (bytes, Required) – Second output. Order does not matter.

Returns

Truth value of two outputs being “close enough”

Return type

bool

time_optimization.run_command(client: client.Client, cmd: str, span: tuple, bytes_to_read: int, wanted: bytes, iterations: int = 1000)

Sends a command through a Client object iteratively. Stops at the first sleep_time where 99% of the iterations return the desired output with 99% accuracy.

Parameters

  • client (Client, Required) – Client object from client

  • cmd (str, Required) – The command to send to the Client

  • span – The span of sleep times to test. Formatted as (start, stop inclusive, number of steps). Ex.

(0.1, 1.0, 10) would test sleep times on the interval [0.1, 0.2, 0.3,…, 1.0] :type span: tuple, Required :param bytes_to_read: The amount of expected bytes needed to read the output. :type bytes_to_read: int, Required :param wanted: The expected output of the command. :type wanted: bytes, Required. :param iterations: The number of times to run the command at each sleep time. :type iterations: int, Optional. Default is 1000. :return: None.

time_optimization.try_time(cmd, sleep, client: client.Client, iterations, bytes_to_read: int, wanted: bytes)

Tries to run the command for a specified number of iterations and uses a specified sleep_time.

Parameters

  • cmd (str, Required) – The command to send to the Client object

  • sleep (float, Required) – The sleep time for the Client object to wait for a response

  • client (Client, Required) – Client object from client

  • iterations (int, Required) – Number of iterations

  • bytes_to_read (int, Required) – The expected number of bytes to read from the output.

  • wanted (bytes, Required) – The expected output from the command.

Returns

The number of outputs from the client that follow the criteria of

close_enough. :rtype: int
